# 预算追踪器

一个简洁的个人预算管理网页应用，支持双货币实时汇率转换。

## 功能特点

- 💰 **月度预算管理** - 设置和追踪每月预算
- 💱 **双货币支持** - 支持多种货币，实时汇率转换
- 📊 **支出记录** - 记录和管理日常支出
- 🎯 **储蓄目标** - 设置储蓄目标并追踪进度
- 🛍️ **愿望清单** - 记录想买的东西及价格
- 📱 **响应式设计** - 支持手机、平板、电脑访问
- 💾 **本地存储** - 数据保存在浏览器本地

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 选择你的主货币和次货币
3. 设置月度预算
4. 开始记录支出、储蓄目标和愿望清单

## 支持的货币

- CNY (人民币)
- USD (美元)
- EUR (欧元)
- GBP (英镑)
- JPY (日元)
- HKD (港币)

## 技术说明

- 纯前端应用，无需服务器
- 使用 localStorage 保存数据
- 使用免费汇率 API 获取实时汇率
- 响应式设计，适配各种设备

## 注意事项

- 数据保存在浏览器本地，清除浏览器数据会丢失记录
- 需要网络连接来获取实时汇率
- 建议定期导出备份数据（可以后续添加导出功能）
