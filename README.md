# 预算追踪器

一个功能完整的个人预算管理应用，支持多设备同步、双货币转换、税费计算等功能。

## 功能特点

- 💰 **月度预算管理** - 设置和追踪每月预算
- 💱 **双货币支持** - 支持多种货币，实时汇率转换（每小时自动更新）
- 📊 **支出记录** - 记录和管理日常支出，保存添加时的汇率
- 🛍️ **愿望清单** - 记录想买的东西及价格，支持税费计算
- 👤 **用户系统** - 注册登录，数据云端存储
- 📱 **响应式设计** - 支持手机、平板、电脑访问
- 🔄 **多设备同步** - 数据保存在服务器，多设备实时同步

## 技术栈

- **前端**: HTML5 / CSS3 / JavaScript
- **后端**: Node.js + Express
- **认证**: JWT
- **存储**: JSON 文件

## 本地开发

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问应用
http://localhost:3000/auth.html
```

## 部署到服务器

```bash
# 运行部署脚本
./deploy-to-server.sh
```

## 支持的货币

- CNY (人民币)
- USD (美元)
- EUR (欧元)
- GBP (英镑)
- JPY (日元)
- HKD (港币)
- CAD (加币)

## 环境变量

创建 `.env` 文件：

```
PORT=3000
JWT_SECRET=your-secret-key
NODE_ENV=production
```
